# 喜茶家厨 TWA APK 构建项目

这个项目用于将喜茶家厨 PWA 网站打包成 Android APK。

## 方法一：使用 GitHub Actions（推荐）

1. 在 GitHub 上创建一个新仓库
2. 将所有文件推送到该仓库
3. GitHub Actions 会自动构建 APK
4. 在 Actions 页面下载构建好的 APK

## 方法二：使用 Android Studio

1. 下载并安装 Android Studio
2. 打开此项目
3. 点击 Build -> Generate Signed Bundle / APK
4. 选择 APK 并签名
5. 下载生成的 APK

## 方法三：使用在线 TWA 生成器

1. 访问 https://appmaker.mercury.io/
2. 输入网站地址：http://47.95.179.240
3. 下载生成的 APK
